# 抖音火花自动续期工具

一款基于Python+Selenium实现的抖音自动化工具，支持自动获取登录Cookie、每日定时给指定好友续火花，无需手动操作

## 📋 功能特性

-   🥠
    **Cookie自动获取**：模拟登录抖音网页版，一键提取登录Cookie，支持持久化使用
-   🔥
    **自动续火花**：配置好友列表后，每日定时给指定好友发送自定义文案，自动续火花
-   📧 **异常邮件提醒**：运行出错时自动发送邮件通知，及时发现问题
-   ⏰ **灵活定时**：支持自定义每日续火花时间，也可立即执行测试
-   🛡️ **防检测策略**：内置浏览器防封配置，模拟真人操作，降低风控风险
-   💬 **自定义文案**：随机发送温馨续火花文案，避免固定内容被风控

## 🖥️ 环境准备

### 1. 安装Python

-   下载并安装Python 3.8+版本：https://www.python.org/downloads/
-   安装时勾选「Add Python to PATH」，确保环境变量配置成功

### 2. 安装依赖包

打开命令提示符（CMD），执行以下命令安装所需依赖：

``` bash
pip install selenium schedule tqdm
```

### 3. 配置Edge驱动

-   查看本地Edge浏览器版本：打开Edge → 设置 → 关于Microsoft Edge
-   下载对应版本的Edge驱动：https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/
-   将驱动文件解压到指定路径（代码中默认路径：`C:\WebDriver\edge\msedgedriver.exe`）

## ⚙️ 配置步骤

### 1. 代码配置（核心）

打开代码文件，找到「配置区」，按说明修改以下内容：

``` python
# -------------------------- 配置区 --------------------------
cookies_list = [] # 填入后续获取的Cookie列表（先留空，运行后复制）
friends_list = ["好友1昵称", "好友2昵称"] # 替换为要续火花的好友昵称（必填）
OFF_ON_Aaiqky_TEXT = True # 是否启用自定义文案（建议开启）
OFF_ON_ERROR_Email = True # 是否启用异常邮件提醒（可选）

# 邮箱配置 [仅当OFF_ON_ERROR_Email=True时需要配置]
my_sender = 'xxx@163.com' # 发件人邮箱（163邮箱为例）
my_pass = '' # 邮箱授权码（不是登录密码！）
my_user = 'xxx@163.com' # 收件人邮箱（可填自己）
# --------------------------------------------------------------------------------
```

### 2. 邮箱授权码获取（可选）

若启用邮件提醒，需获取邮箱授权码（以163邮箱为例）： 1. 登录163邮箱 →
设置 → POP3/SMTP/IMAP 2. 开启SMTP服务 → 生成授权码 → 复制到`my_pass`字段

### 3. 元素类名适配（可选）

若后续抖音网页版更新导致按钮定位失败，需更新`CLASS_NAMES`中的类名：

## 🚀 使用教程

### 第一步：获取Cookie

1.  确保`cookies_list = []`（留空），运行代码：

``` bash
py 你的代码文件名.py
```

2.  代码会自动打开Edge浏览器并跳转到抖音登录页
3.  用抖音APP扫码登录 → 登录成功后，控制台会打印Cookie列表
4.  复制打印的Cookie列表，粘贴到代码中的`cookies_list`字段

### 第二步：配置好友列表

将`friends_list`替换为需要续火花的好友昵称（**必须和抖音私信列表中的昵称完全一致**）：

``` python
friends_list = ["张三", "李四"] # 示例：替换为你的好友昵称
```

### 第三步：运行续火花程序

1.  保存修改后的代码，重新运行：

``` bash
py spark_continuation.py
```

2.  程序会加载Cookie并验证登录状态：
    -   若提示「Cookie有效, 登录成功」，说明登录正常
    -   若提示「Cookie无效」，需重新执行第一步获取Cookie
3.  输入每日续火花时间（默认22:00），按回车确认
4.  程序会先执行一次续火花测试，之后在指定时间自动运行

## ❗ 常见问题

### Q1：Cookie获取失败/无效？

-   解决方案：
    1.  关闭所有Edge浏览器窗口，重新运行代码
    2.  登录时确保浏览器全屏，扫码后等待页面完全加载
    3.  Cookie有效期约1-7天，失效后需重新获取

### Q2：找不到好友/点击私信按钮失败？

-   解决方案：
    1.  检查`friends_list`中的昵称是否和抖音私信列表完全一致（区分大小写/特殊符号）
    2.  更新`CLASS_NAMES`中的类名（F12开发者工具定位最新类名）
    3.  确保抖音网页版已切换到「私信」标签页

### Q3：发送消息失败/发送按钮不显示？

-   解决方案：
    1.  程序内置了「模拟真人输入」逻辑，无需手动操作
    2.  延长代码中的`time.sleep()`等待时间（适配网络慢的情况）
    3.  关闭抖音网页版的弹窗/广告，避免遮挡元素

### Q4：邮件提醒无法发送？

-   解决方案：
    1.  确认发件人邮箱已开启SMTP服务
    2.  检查`my_pass`是否为授权码（不是登录密码）
    3.  确保网络能访问SMTP服务器（163邮箱：smtp.163.com:465）

## 📌 注意事项

1.  请勿频繁运行程序（建议每日1次），避免抖音风控限制
2.  Cookie包含登录信息，请勿泄露给他人
3.  若抖音网页版更新导致程序失效，需更新元素定位类名
4.  运行时保持网络稳定，避免中途断网导致任务失败
5.  建议在服务器/挂机电脑上运行，确保定时任务正常执行
